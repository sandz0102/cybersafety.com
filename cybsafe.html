Easy Security Habits

Better Passwords

Think of passwords like toothbrushes - don't share them and change them regularly. Mix uppercase, lowercase, numbers and symbols.

Two-Step Verification

It's like a second lock. Even if someone gets your password, they can't get in without your phone.

Suspicious Links

If a link looks fishy, it probably is. Hover to see where they go before clicking.

Password Managers: Your Digital Safe

Why bother?

  • Remembers all passwords
  • Creates strong passwords
  • Warns about weak/reused passwords

How They Work

Imagine a super secure notebook with one key (master password). All others are inside.

Automatically fills passwords. No more sticky notes!

Getting Started

  1. Pick a manager (free options exist)
  2. Create one strong master password
  3. Let it generate and store passwords
  4. Install browser extension and phone app

Common Online Threats

Phishing Scams

Fake emails/texts tricking you for passwords. Always verify sender addresses and never click suspicious links.

Malware

Bad software stealing info or damaging devices. Only download from trusted sources and keep antivirus updated.

Public Wi-Fi Risks

Unprotected use can be overheard. Use VPNs for sensitive activities on public networks.
